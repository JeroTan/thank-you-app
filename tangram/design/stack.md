# Tech-Stack

- **Core Framework**: Astro JS (Latest)
- **UI Rendering**: React JS (Latest) for complex SVG DOM manipulation and reactivity.
- **Styling**: Tailwind CSS (Latest) for HTML overlays, scaffolding, and utility classes.
- **Animation & Physics**: Anime.js for dragging physics, oscillating strings, and smooth transitions.
- **Environment & Hosting**: Cloudflare Workers (Native support via `@astrojs/cloudflare`).
- **Data Source**: A mocked `.ts` file serving as the initial API response structure.
- **Code Formatting**: Prettier configured with Astro and Tailwind CSS plugins to enforce consistent code style.
