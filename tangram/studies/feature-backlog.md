# Feature Backlog

*These are logical next steps beyond the current MVP scope:*
- **Search/Filter:** Ability to search for a specific person's name and auto-pan the camera to their marker.
- **Real-time Updates:** Connecting the map to a live WebSocket or real API to see "thank yous" animate into existence.
- **Detailed Profile View:** Clicking a marker opens a modal or side panel with a list of exactly who thanked them and when.
- **Performance Toggle:** An option to disable physics or complex gradients on lower-end mobile devices.
- **Export/Share:** Ability to snapshot a specific view of the canvas to share on social media.

## Post-00009 Product Backlog
- **Search/Auto-pan Refinement:** Expand simple marker search into richer filters by count, connection type, and pinned state.
- **Responsive Card Fallback:** Provide a static or simplified marker card grid for small screens where dense canvas interaction is not ideal.
- **Live Updates:** Connect to a real API or WebSocket so thank-you relationships can animate into the map as they arrive.
- **Dense Dataset Controls:** Add clustering, level-of-detail string rendering, and an explicit performance mode for larger populations.
- **Shareable Map Views:** Export a snapshot or share a URL encoding selected marker, zoom, and pan position.
