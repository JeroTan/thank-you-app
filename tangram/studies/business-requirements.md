# Business Requirements

- An infinite, pannable, and zoomable map with a repeating green sprite background.
- Dynamic SVG markers representing users, scaled proportionally based on the number of "thank yous" they've received.
- Markers display user pictures or initials, styled with a specific frame color.
- Polylines ("strings") connecting related markers.
- Strings exhibit oscillating/swinging physics when markers are dragged.
- Mutual (duplex) thanks use two-color gradients on strings.
- Click interactions to highlight a marker and its connections while dimming others.
- A mocked `.ts` dataset representing the API response.
